# Machine Learning Algorithms

Implementation of fundamental machine learning algorithms developed during the Foundations of Machine Learning course at IIT Madras.

## Algorithms

- PCA
- Kernel PCA
- K-Means
- Spectral Clustering
- EM Algorithm
- Linear Regression
- Ridge Regression
- Gradient Descent
- Stochastic Gradient Descent

## Applications

- Dimensionality Reduction
- Clustering
- Regression
- Spam Classification
